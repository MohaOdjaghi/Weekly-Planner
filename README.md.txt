# Weekly Planner

A clean and simple weekly planner web app to organize your tasks with color-coded categories.

## Features

- Weekly planner layout
- Color categories for task types
- Add and save tasks for each day of the week
- Tasks saved locally in browser (no backend required)
- Start a new week with a fresh planner

## How to Use

1. Select a day of the week.
2. Add your tasks and assign a color category.
3. Click "Save" to store your tasks in local storage.
4. Start a new week whenever you want â€” previous week's tasks are saved separately.
